# OULAD Dataset

This project utilizes the **Open University Learning Analytics Dataset (OULAD)**. Due to file size constraints, the raw data is not included in this repository.

## How to Download
1. Visit the official OULAD repository: [Open University Learning Analytics Dataset](https://analyse.kmi.open.ac.uk/open_dataset) (or via [Kaggle](https://www.kaggle.com/datasets/anlthms/ou-student-learning-and-demographics)).
2. Download the `.zip` file containing the dataset.
3. Extract the contents.

Your `data/` folder should look like this before running the preprocessing scripts:
- `data/assessments.csv`
- `data/courses.csv`
- `data/studentAssessment.csv`
- `data/studentInfo.csv`
- `data/studentRegistration.csv`
- `data/studentVle.csv`
- `data/vle.csv`
